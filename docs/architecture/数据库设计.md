# 数据库设计约定

## 通用字段

业务表必须包含：

- `id`
- `create_time`
- `update_time`
- `create_by`
- `update_by`
- `deleted`

## 命名约定

- 表名使用小写下划线。
- 字段名使用小写下划线。
- Java DO 使用 UpperCamelCase，以 `DO` 结尾。
- Mapper XML 文件名不加 `I` 前缀。

## 逻辑删除

重要业务表不做物理删除，统一使用 `deleted` 字段。

## 任务表

文档处理任务必须记录：

- 任务 ID。
- 文档 ID。
- 文档版本 ID。
- 当前阶段。
- 当前状态。
- 重试次数。
- 失败原因。
- 回调时间。
