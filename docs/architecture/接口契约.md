# 接口契约

## 通用返回

所有接口统一返回 `Result<T>`。

分页接口返回 `PageResult<T>`。

## 对象边界

- Controller 接收 Request。
- Controller 返回 Response。
- Service 接收 Request。
- Mapper 接收 Query。
- Mapper 返回 DO。
- Convert 负责 Request、Query、DO、Response 转换。

## 分页

分页查询使用 MyBatis-Plus：

```java
IPage<XxxDO> selectPage(Page<XxxDO> page, @Param("query") XxxPageQuery query);
```

## 错误处理

- 参数错误返回明确错误码。
- 业务错误抛出业务异常。
- 系统错误统一由全局异常处理器兜底。
