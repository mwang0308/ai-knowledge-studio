# 文档解析

## 目标

将上传文档解析为结构化内容，形成可分片、可预览、可追踪的处理结果。

## 支持格式

- PDF
- Word
- Excel
- Markdown
- TXT
- CSV

## 主流程

```text
文档上传
→ 创建处理任务
→ RabbitMQ 投递任务
→ Python 消费任务
→ 解析文档结构
→ 生成结构块
→ 生成分片
→ 写入 ES / Milvus
→ 回调 Java 更新状态
```

## 必须记录

- 文档 ID。
- 文档版本。
- 处理任务 ID。
- 当前阶段。
- 当前状态。
- 失败原因。
- 重试次数。
- 解析器名称。

## 异常处理

- 解析失败必须记录失败状态和错误信息。
- 支持人工重试。
- 重试时必须保留原任务记录和新任务记录之间的关系。
