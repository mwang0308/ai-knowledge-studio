# 前端编码规范

## 技术栈

- Vue 3
- TypeScript
- Vite
- Element Plus
- Pinia
- Vue Router
- Axios

## 目录约定

- 页面放在 `views/`。
- 可复用组件放在 `components/`。
- 接口请求放在 `api/`。
- 类型定义放在 `types/`。
- 状态管理放在 `stores/`。

## 交互约定

- 管理端界面以清晰、紧凑、可扫描为主。
- 表格、筛选、状态标签、操作按钮必须清晰。
- 上传、解析、发布等流程必须展示状态和错误原因。

## 日志和错误

- 前端请求失败要展示明确错误信息。
- 不在控制台输出敏感数据。
